# 02 專業證照輪播版面與 UI 呈現
# 02 Certificate Carousel Layout and UI

## 使用者故事 User Story

**作為 As** 網站訪客  
**我想要 I want** 在 PC 與 Mobile 上看到一致風格的專業證照輪播區塊  
**以便 So that** 能直覺地瀏覽所有證照素材並了解其名稱與位置。

## 輸入格式 Input

- 一組 `CertificateMediaItem[]`，已包含：
  - `previewImageUrl`
  - `name`
  - `mediaType`
- 顯示環境資訊：
  - 視窗寬度 / 斷點（PC / Mobile）。
- 可選 input：
  - 初始 active index（預設 0）。

## 輸出格式 Output

- 一個前端 UI 元件（例如 `CertificateCarousel` React component）：
  - PC 版：
    - 當前主素材視覺置中，左右可見相鄰預覽。
    - 左右箭頭控制按鈕。
  - Mobile 版：
    - 仍保留主素材閱讀焦點並可顯示相鄰預覽。
    - 左右箭頭控制與滑動支援。
  - 通用：
    - 所有預覽圖以 4:5 比例呈現。
    - 主素材下方顯示素材名稱。
    - 區塊下方有輪播進度指示器。

## 任務內容 Task Description

- 實作一個負責 UI 呈現的輪播元件，僅關注版面配置與視覺層。
- 依據 Tailwind / 設計稿完成：
  - 統一的 4:5 預覽圖容器（含裁切邏輯與圓角）。
  - 主素材與相鄰素材的位置與大小差異。
  - RWD 版面切換（PC / Mobile 構圖可不同）。
- 暴露簡單的 props：
  - `items: CertificateMediaItem[]`
  - `activeIndex: number`
  - `onChangeActive(index: number): void`

## 驗收條件 Acceptance Criteria

- [ ] PC 版與 Mobile 版在對應斷點呈現不同布局，但皆保留主素材與相鄰預覽。
- [ ] 所有預覽圖在 UI 上維持 4:5 比例與一致的圓角與裁切方式。
- [ ] 主素材下方正確顯示當前素材名稱。
- [ ] 區塊下方輪播進度指示器與 `activeIndex` 同步。
- [ ] 左右箭頭按鈕可以透過 props 回呼通知外部更新 active index。
- [ ] 不涉及開啟彈窗與實際內容顯示（僅 UI & navigation shell）。

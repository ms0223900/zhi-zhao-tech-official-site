# 02 專業證照輪播版面與 UI 呈現
# 02 Certificate Carousel Layout and UI

## 使用者故事 User Story

**作為 As** 網站訪客  
**我想要 I want** 在 PC 與 Mobile 上看到一致風格的專業證照輪播區塊  
**以便 So that** 能直覺地瀏覽所有證照素材並了解其名稱與位置。

## 輸入格式 Input

- 一組 `CertificateMediaItem[]`，已包含：
  - `previewImageUrl`
  - `name`
  - `mediaType`
- 顯示環境資訊：
  - 視窗寬度 / 斷點（PC / Mobile）。
- 可選 input：
  - 初始 active index（預設 0）。

## 輸出格式 Output

- 一個前端 UI 元件（例如 `CertificateCarousel` React component），前端輪播行為以 `slick-carousel` / `react-slick` 實作，整體設定與此專案現有的 Carousel 元件（例如首頁 Hero / Testimonials 等輪播）保持一致：
  - 內部使用 `react-slick` 的 `Slider` 元件，並套用專案既有的 slick 樣式（共用的箭頭、dots、滑動動效等）。
  - PC / Mobile 皆透過 `react-slick` 的 `responsive` 設定切換顯示張數與中心對齊行為（以 `centerMode` + `centerPadding` 達成主素材置中、左右露出相鄰預覽）。
  - 左右箭頭使用與其他 Carousel 共用的 React component（例如 `CarouselPrevButton` / `CarouselNextButton`），並透過 `slick-prev` / `slick-next` className 套用樣式。
  - 輪播進度指示器（dots）沿用專案內客製化 slick dots 的樣式，並與 `activeIndex` 同步。
  - 若未來專案有全域 Carousel wrapper（例如 `BaseCarousel`），`CertificateCarousel` 應優先組合 / 包裝該元件，而非自行直接操作 slick 實例。
  - PC 版：
    - 當前主素材視覺置中，左右可見相鄰預覽。
    - 左右箭頭控制按鈕。
  - Mobile 版：
    - 仍保留主素材閱讀焦點並可顯示相鄰預覽。
    - 左右箭頭控制與滑動支援。
  - 通用：
    - 所有預覽圖以 4:5 比例呈現。
    - 主素材下方顯示素材名稱。
    - 區塊下方有輪播進度指示器。

## 任務內容 Task Description

- 實作一個負責 UI 呈現的輪播元件，僅關注版面配置與視覺層。
- 依據 Tailwind / 設計稿完成：
  - 統一的 4:5 預覽圖容器（含裁切邏輯與圓角）。
  - 主素材與相鄰素材的位置與大小差異。
  - RWD 版面切換（PC / Mobile 構圖可不同）。
- 暴露簡單的 props：
  - `items: CertificateMediaItem[]`
  - `activeIndex: number`
  - `onChangeActive(index: number): void`

## 驗收條件 Acceptance Criteria

- [x] PC 版與 Mobile 版在對應斷點呈現不同布局，但皆保留主素材與相鄰預覽。
- [x] 所有預覽圖在 UI 上維持 4:5 比例與一致的圓角與裁切方式。
- [x] 主素材下方正確顯示當前素材名稱。
- [x] 區塊下方輪播進度指示器與 `activeIndex` 同步。
- [x] 左右箭頭按鈕可以透過 props 回呼通知外部更新 active index。
- [x] 不涉及開啟彈窗與實際內容顯示（僅 UI & navigation shell）。

## 實作說明 Implementation Notes

- **程式碼位置**：
  - `src/components/certificate/CertificateCarousel.tsx`：專業證照輪播元件
  - `src/components/certificate/CertificateSection.tsx`：專業證照區塊（含標題與輪播）
  - `src/components/certificate/index.ts`：匯出
- **匯入範例**：
  ```typescript
  import { CertificateCarousel, CertificateSection } from "@/components/certificate";
  import type { CertificateMediaItem } from "@/types/certificate-media";
  ```
- **RWD**：PC（≥768px）3 欄、Mobile（<768px）2 欄，皆使用 `centerMode` + `centerPadding` 達成主素材置中。
- **展示頁**：`/certificate-demo` 可供開發時預覽輪播效果。

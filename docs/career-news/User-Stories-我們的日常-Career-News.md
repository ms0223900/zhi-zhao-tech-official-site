## 使用者角色定義

### 角色 1：求職者 (Job Seeker)
- **描述**：正在尋找工作機會的潛在求職者
- **目標**：了解公司文化、工作環境與員工生活
- **使用情境**：評估是否適合該公司文化

### 角色 2：現有員工 (Current Employee)
- **描述**：智兆科技的現職員工
- **目標**：查看公司活動、日常分享與最新消息
- **使用情境**：了解公司動態與參與活動

### 角色 3：訪客 (Visitor)
- **描述**：一般網站訪客
- **目標**：了解智兆科技企業文化與品牌形象
- **使用情境**：瀏覽公司資訊

---

## P0 優先級 - 核心功能（必須實現）

### US-002：查看「我們的日常」文章列表
**作為** 求職者  
**我想要** 瀏覽「我們的日常」文章列表  
**以便** 了解公司內部日常活動與文化

**驗收條件**：
- [x] 區塊標題顯示「我們的日常」與「Career News」
- [x] 每篇文章顯示標題、發布日期
- [x] 每篇文章前有藍色箭頭圖示，表示可點擊
- [x] 文章列表垂直排列，每篇文章間有適當間距
- [x] 預設每頁顯示 3 篇文章
- [x] 標題可點擊，連結至文章詳情頁
- [x] 日期格式為 YYYY-MM-DD
- [x] 響應式設計在各種裝置上正常運作

**優先級**：P0  
**相關功能**：文章列表區塊（2.1.2）

---


### US-004：使用分頁功能瀏覽文章
**作為** 訪客  
**我想要** 使用分頁功能瀏覽多頁文章  
**以便** 有效率地查看所有文章

**驗收條件**：
- [x] 顯示當前頁碼格式：`第 {current}/{total} 頁`（例如：第 1/2 頁）
- [x] 提供「上一頁」按鈕，用於切換到上一頁
- [x] 提供「下一頁」按鈕，用於切換到下一頁
- [x] 在第一頁時，「上一頁」按鈕為禁用狀態
- [x] 在最後一頁時，「下一頁」按鈕為禁用狀態
- [x] 分頁切換流暢無卡頓

**優先級**：P0  
**相關功能**：分頁控制區塊（2.1.3）

---

### US-005：閱讀文章詳情
**作為** 求職者  
**我想要** 點擊文章標題進入詳情頁閱讀完整內容  
**以便** 深入了解公司活動與文化

**驗收條件**：
- [x] 點擊文章標題可進入詳情頁
- [x] 詳情頁顯示完整文章內容
- [x] 文章內容包含標題、副標題、發布日期、封面圖片、正文，可以直接沿用「複製」最新消息頁面的組件來使用
- [x] 有返回列表按鈕，可以返回上一頁（/careers）
- [x] 文章內容正確渲染（支援 Markdown 格式），文章內容 SCHEMA 同最新消息頁面
- [x] 響應式設計在各種裝置上正常顯示

**優先級**：P0  
**相關功能**：文章詳情頁（2.4.1）

---


## 響應式設計需求

### US-008：手機版瀏覽體驗
**作為** 使用手機的求職者  
**我想要** 在手機上舒適地瀏覽文章列表  
**以便** 隨時隨地了解公司文化

**驗收條件**：
- [ ] 文章列表在手機上垂直排列
- [ ] 文字大小適中，易於閱讀
- [ ] 按鈕大小適合觸控操作
- [ ] 分頁控制在小螢幕上正常顯示
- [ ] 無水平滾動條

**優先級**：P0（響應式需求）

---

### US-009：平板版瀏覽體驗
**作為** 使用平板的訪客  
**我想要** 在平板上獲得良好的瀏覽體驗  
**以便** 充分利用螢幕空間

**驗收條件**：
- [ ] 佈局在平板尺寸（768px - 1024px）上正常顯示
- [ ] 文字與間距適中
- [ ] 觸控操作流暢

**優先級**：P0（響應式需求）
